Archivematica-ArchivesSpace DIP Upload Integration

An Archivematica DIP

Use case

  • We needed to link a large volume of digital objects to existing description
  • We needed to create rights statements that are machine actionable
  • We needed the process to be scalable

What does this integration do?

  • Rights information and technical metadata is passed from Archivematica to ArchivesSpace through automated processes
  • This automated recording of DIP information in ArchivesSpace can be done either through manual linking or through filename matching

PREMIS Rights Statement Mapping

  • Publish: Set as TRUE or FALSE based on "publish" act.
  • Restrictions?: Set as TRUE or FALSE based on based on "disseminate" act.
  • Conditions Governing Access note: Derive from PREMIS rightsGrantedNote based on "disseminate" act. (This can also be set in the Archivematica administration settings.)
  • Conditions Governing Use note: Derive from PREMIS rightsGrantedNote based on "publish" act. (This can also be set in the Archivematica administration settings.)
